## 3. 변경된 API / 정책(있으면)

- 신규 API:
  - `POST /api/v1/collections`
  - `GET /api/v1/collections?mine={boolean}`
  - `GET /api/v1/collections/{collectionId}`
  - `POST /api/v1/collections/{collectionId}/items`
  - `GET /api/v1/courses/{courseId}/highlights`
  - `POST /api/v1/courses/{courseId}/highlights`
- 인증 정책:
  - Collection 생성/코스 담기, Highlight 생성은 Bearer JWT 필요
  - `mine=true` 컬렉션 목록은 로그인 사용자만 조회 가능
- 공개 범위 정책:
  - Collection: `private / unlisted / public`
  - Highlight: `public / private`

# Notion 가져오기(Import) 가이드

이 폴더(`notion-import/`)는 Notion에서 **폴더째 Import**하면, 폴더 구조가 그대로 **페이지/서브페이지**로 생성되도록 구성했습니다.

## 1) Notion으로 가져오기

1. Notion 좌측 사이드바에서 가져올 워크스페이스/페이지로 이동
2. `Import` 선택
3. `Markdown & CSV` 선택
4. 이 폴더(`notion-import/`)를 통째로 드래그/선택

## 2) 권장 설정(프로젝트 느낌으로 정리)

- 최상위 페이지(폴더명)를 프로젝트 홈으로 사용
- 사이드바에 고정(Pin)
- 각 섹션 페이지에 `Table of contents` 블록을 추가(선택)

## 3) 원본(SSOT)

원본 설계는 레포의 `설계/` 폴더입니다. 이 Notion Import본은 **가독성/구조화 목적**이며, 결정/계약 변경은 원본 문서를 먼저 갱신하는 방식(SSOT)을 권장합니다.

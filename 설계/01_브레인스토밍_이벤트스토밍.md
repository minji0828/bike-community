## 도메인 가정 (초기)

- 서비스: 코스(경로) 기반 자전거 산책 지도 + 코스 공유 + POI(우선: 화장실, vNext: 카페/편의)
- 길안내 로드맵:
  - MVP: 코스 따라가기 가이드(진행률/이탈 감지) = 레벨 2
  - 장기: 턴바이턴(음성/재탐색/교차로 안내) = 레벨 4
- 데이터: 위치(실시간/이력), 코스 경로(LineString), POI(Point), (추후) 리뷰/댓글
- 외부 연동: 공공데이터(화장실), 지도 SDK, (추후) 길찾기 API/내비 SDK

## 이벤트스토밍 표기(권장)

- Domain Event: 과거형 문장 (예: `라이딩이_저장되었다`)
- Command: 동사형 (예: `라이딩_저장하기`)
- Aggregate: 상태/불변식의 경계 (예: `Riding`)
- Policy/Saga: 이벤트->커맨드 트리거 규칙
- Read Model/View: 조회 전용 모델
- External System: 외부 API/배치/스케줄러

## Aggregates / Entities 후보

- `User`
- `UserLocation` (이력/현재)
- `Riding`
- `Poi`
- `Course`
- `(추후) CourseVersion` (코스 수정/리뷰 일관성을 위해)
- `(추후) CourseReview`, `Comment`, `Like`

불변식/규칙 후보:

- 한 사용자는 `현재 위치(isCurrent=true)`가 최대 1개
- 라이딩 경로 좌표는 `(lon, lat)` 순서로 저장/계산
- POI는 `externalId`로 외부 데이터와 매핑
- 코스 공유 범위는 `public/unlisted/private` 중 하나
- 코스 따라가기는 레벨 2 기준(코스 선 + 진행률/이탈)이며, 레벨 4는 별도 단계로 분리

## Read Model 후보

- 사용자 현재 위치 조회
- 사용자 위치 이력(페이지네이션)
- 주변 POI(반경)
- 경로 주변 POI(라인버퍼)
- 라이딩 상세(거리/시간/경로)
- 코스 목록/상세
- 공유 코스 상세(공개 범위 반영)
- 코스 따라가기 화면용 요약(거리, 예상시간 등)

## Policy / 자동화 후보

- `poi.fetch.onstartup=true`면 앱 시작 시 POI 동기화 수행
- 스케줄러(추후): 매일 새벽 증분 동기화
- 데이터 보관(추후): 위치 이력 30일 지난 데이터 삭제
- (추후) 코스 생성 시 경로 단순화/요약(거리/고도 등) 계산
- 코스 생성/업로드 시: distance/bbox/loop 즉시 산출
- 코스 생성/업로드 후: amenitiesSummary(코스 주변 POI 요약) 비동기 산출
- POI 동기화 후: featured/최근 코스 우선으로 메타데이터 재계산
- (추후) 인기 코스/조회수 집계는 이벤트 기반 카운터로 분리

- `User/Identity` (계정, 디바이스)

- `Riding` (라이딩 기록)

- (추후) `Community` (리뷰/댓글/좋아요)

## 유비쿼터스 언어 (초기 용어집)

| 용어 | 의미 | 비고 |
|---|---|---|
| POI | Point of Interest | MVP: 화장실 우선, vNext: 카페/편의 확장 |
| 현재 위치 | 최신 위치 1건 | `isCurrent=true` |
| 위치 이력 | 과거 위치 목록 | retention 정책 필요 |
| 증분 동기화 | 변경분만 반영 | 외부 API 특성에 따라 정의 |
| 코스 | 공유 가능한 경로 콘텐츠 | 기본 단위(UGC) |
| 코스 따라가기 | 코스 선을 따라가는 안내 | 레벨 2 |
| 턴바이턴 | 단계별 교차로 안내 내비 | 레벨 4 |

## 오픈 이슈 / 질문

- 인증/권한은 MVP에 포함할 것인가? (현재 Security는 all permit)
- 위치 업데이트 빈도/배치 크기/정밀도는 어떻게 둘 것인가?
- POI 동기화 실패 시 재시도/알림/회복 전략은?
- 개인정보(정확한 위치) 로깅/저장 정책은?
- 코스 따라가기(레벨 2)에서 이탈 기준(미터)과 히스테리시스는?
- 턴바이턴(레벨 4)은 외부 내비 앱 호출 vs 앱 내 내비 SDK vs 자체 라우팅 중 무엇으로 갈지?

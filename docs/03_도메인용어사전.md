# 03_도메인용어사전

- 작성일: 2026-03-10
- 상태: 활성 SSOT

## 1. 용어 사전

| 용어 | 정의 | 비고 |
|---|---|---|
| User | 서비스 사용 주체 | 로그인 사용자 |
| AuthProvider | 로그인 제공자 | 현재 `KAKAO`만 사용 |
| Riding | 실제 주행 기록 원본 | path 좌표를 저장 |
| Course | 저장/공유 가능한 코스 콘텐츠 | 핵심 도메인 |
| Featured Course | 운영이 우선 노출하는 코스 | 홈 진입점 |
| POI | Point of Interest | MVP1은 화장실 중심 |
| Nearby POI | 현재 위치 기준 편의시설 조회 | 홈에서 사용 |
| Along-route POI | 경로 기준 편의시설 조회 | 코스/라이딩에서 사용 |
| Guide L2 | Polyline + 현재 위치 + 진행률 + 이탈 감지 | 턴바이턴 아님 |
| shareId | 공유 코스 식별자 | 공개/링크공유 코스 조회에 사용 |
| Tag | 코스를 설명하는 짧은 키 | 예: river, beginner |
| Warning | 코스 위 주의 정보 | 구조화된 경고 |
| Course Collection | 여러 코스를 묶는 여행 단위 콘텐츠 | P1 이후 |
| Course Highlight | 코스 위 포인트 메모 | P1 이후 |
| Course Meetup | 코스 기반 모임 | P1 이후 |

## 2. 반드시 통일할 용어

- 코스 = `Course`
- 라이딩 기록 = `Riding`
- 기본 제공 코스 = `Featured Course`
- 코스 따라가기 = `Guide L2`
- 공유 링크 식별자 = `shareId`
- 코스모임 = `Course Meetup`

## 3. 금지 용어

- `Course` / `Route` / `Track` 혼용 금지
  - 저장/공유 가능한 콘텐츠는 항상 `Course`
- `User` / `Member` 혼용 금지
- `Meetup` / `Event` / `Gathering` 혼용 금지
  - 현재 도메인에서는 `Course Meetup`
- `lat/lng` 와 `lat/lon`을 API 경계 없이 혼용 금지
  - 프론트 뷰 모델은 `lat/lng`
  - API DTO는 `lat/lon`

## 4. 핵심 관계

- User는 카카오 인증을 통해 로그인할 수 있다.
- User는 Riding을 저장할 수 있다.
- Riding은 Course 생성의 원본이 될 수 있다.
- Course는 path, visibility, sourceType, verifiedStatus를 가진다.
- Course는 shareId로 공개/링크공유 조회가 가능할 수 있다.
- Course는 Tag와 Warning을 가진다.

# MVP1 E2E 체크리스트 (실서비스 검증)

## A. 공통/접속
- [ ] `https://<domain>/` 접속 성공
- [ ] `GET /api/v1/health` 응답 code=200

## B. 코스/라이딩
- [ ] Featured 목록 조회
- [ ] 코스 상세(경로 + warnings + amenities) 표시
- [ ] 라이딩 기록 저장
- [ ] from-riding 코스 생성
- [ ] 공유 링크 발급 및 public 조회

## C. 인증/댓글(MVP2)
- [ ] 카카오 코드 교환 성공 (토큰 수신)
- [ ] 댓글 작성 성공
- [ ] 본인 댓글 삭제 성공
- [ ] 타인 댓글 신고 성공

## D. 모임/채팅(MVP3)
- [ ] 코스 모임 생성
- [ ] 모임 참여/탈퇴
- [ ] STOMP 연결 성공 (`/ws-stomp`)
- [ ] 채팅 메시지 송신/수신(2클라이언트 교차 검증)

## E. 운영 API
- [ ] 태그 조회/생성/수정/비활성화
- [ ] 코스 경고 조회/생성/삭제
- [ ] 메타데이터 재계산 호출 성공

## F. 실패/복구
- [ ] 잘못된 토큰으로 write API 요청 시 401 ApiResponse
- [ ] 권한 없는 요청 시 403 ApiResponse
- [ ] Blue/Green 롤백 절차 수행 및 복구 확인
